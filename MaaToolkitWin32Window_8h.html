MAA_TOOLKIT_API MaaSize MaaToolkitFindWindow ( MaaStringView  class_name,
MaaStringView  window_name 
)

Find a win32 window by class name and window name.

This function finds the function by exact match. See also MaaToolkitSearchWindow().

Parameters
class_nameThe class name of the window. If passed an empty string, class name will not be filtered.
window_nameThe window name of the window. If passed an empty string, window name will not be filtered.
Returns
MaaSize The number of windows found that match the criteria. To get the corresponding window handle, use MaaToolkitGetWindow().

Regex search a win32 window by class name and window name.

This function searches the function by regex search. See also MaaToolkitFindWindow().

Parameters

window_nameThe window name regex of the window. If passed an empty string, window name will not be filtered.
Returns
